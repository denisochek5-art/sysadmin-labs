## Кто я
Имя: Denis  
Цель: стать junior sysadmin / DevOps (стажёр)

## Куда иду
Хочу за 90 дней освоить базовые навыки администрирования Linux, сети, Git, автоматизацию и мониторинг.

## Что делаю 90 дней (план)
### Дни 1–30: база
- Linux: файловая система, права, пользователи, systemd, логи
- Сеть: IP, DNS, DHCP, TCP/UDP, SSH
- Git/GitHub: репозитории, коммиты, ветки, PR
- Практика: 10–15 мини-лабораторных

### Дни 31–60: администрирование
- Bash: скрипты для рутины
- Сервисы: nginx, sshd, cron, backup
- Безопасность: ключи SSH, sudo, firewall (ufw/iptables)
- Практика: поднять сервер и задокументировать

### Дни 61–90: приближение к продакшену
- Мониторинг: метрики/логи (Prometheus/Grafana или аналоги)
- Контейнеры: Docker основы
- Автоматизация: Ansible основы
- Итог: 1 проект + отчёт в репозитории
